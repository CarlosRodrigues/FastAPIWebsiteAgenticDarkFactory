# Bootstrap 5 & WCAG 2.1 Standards
1. **Grid System**: Use Bootstrap's `container`, `row`, and `col-*` classes exclusively.
2. **Accessibility**: All buttons must have `aria-label` if they lack text. Form inputs need `<label>`.
3. **Color Contrast**: Only use standard contextual classes (`text-primary`, `bg-dark`).
4. **Interactivity**: Use Bootstrap's data attributes (`data-bs-toggle`); avoid custom JS for standard UI.
